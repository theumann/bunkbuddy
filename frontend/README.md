# Frontend – Bunkbuddy

Next.js (App Router) frontend for the Bunkbuddy application.

## Tech

-   Next.js (App Router)
-   TypeScript
-   Tailwind CSS

## Development

```bash
npm installnpm run dev
```

App runs at [http://localhost:3000](http://localhost:3000)The frontend expects the backend API at [http://localhost:4000](http://localhost:4000)
