# Frontend - Bunkbuddy

Next.js (App Router) frontend for the Bunkbuddy application.

## Tech

-   Next.js (App Router)
-   TypeScript
-   Tailwind CSS

## Development

```bash
npm install
```

App runs at `http://localhost:3000`.
The frontend expects the backend API at `http://localhost:4000`.

## Automated Testing

For the frontend (e2e), we are using Playwright. Commands are in the repo level README file (../README from here).
